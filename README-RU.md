## Зачем это приложение

Большинство таск-менеджеров сделаны для нейротипичных людей: длинные списки, глубокие иерархии, еженедельные ревью, сложные настройки. Для мозга с ADHD — это ловушка. Ты тратишь больше времени на организацию задач, чем на их выполнение. Система сама становится прокрастинацией.

**Kanzen Board** сочетает два подхода, каждый из которых доказанно работает при ADHD:

- **Канбан** — всё видно сразу, на одном экране, в виде карточек на доске. Никаких вложенных меню, никаких скрытых бэклогов.
- **Кайдзен** (改善, «непрерывное улучшение») — у каждой задачи есть поле *Impact*. Одно предложение: зачем это важно? Если не можешь ответить — задача, скорее всего, не нужна.

Название **Kanzen** (完全) по-японски означает «завершённый» — задача не сделана, пока она не перемещена, не отслежена и её польза не зафиксирована.

## Почему канбан + кайдзен работает при ADHD

### Мозг с ADHD должен *видеть* работу, чтобы её делать

С глаз долой — из ума вон. Это буквально. Список задач в приложении, которое ты не открываешь, не существует. Доска на экране с карточками, которые можно перетащить — существует. Канбан делает каждую задачу физически присутствующей.

[Исследования подтверждают](https://leantime.io/navigating-adhd-as-a-project-manager/): визуальная организация — одна из самых эффективных стратегий при ADHD. Карточки, колонки и цвета задействуют визуальные сильные стороны мозга с ADHD, снижая когнитивную нагрузку от необходимости помнить, что нужно сделать.

### Перетаскивание = дофамин

Мозг с ADHD хронически испытывает нехватку дофамина — нейромедиатора, отвечающего за мотивацию, доведение дел до конца и ощущение, что что-то было *стоящим*. Перемещение карточки из **In Progress** в **Done** — это физический жест, который даёт небольшой, но реальный дофаминовый всплеск. [Сообщество r/ADHD на Reddit называет это](https://www.reddit.com/r/ADHD/comments/z7i15a/having_a_kanban_probably_saved_my_job/) «взломом дофаминовой системы». Это не трюк — так работает мозг.

### WIP-лимит останавливает переключение между задачами

Один из самых разрушительных паттернов ADHD — переключаться между задачами, не завершая ни одну. Основное правило канбана — ограничить количество задач в работе — даёт структуру, чтобы противостоять этому импульсу. Колонка **In Progress** — это ограничение: если там уже есть карточка, сначала заверши её.

### Кайдзен убивает ловушку «занятости ради занятости»

Мозг с ADHD реагирует на интерес, а не на важность. Это легко приводит к тому, что день заполнен мелкими задачами, которые *ощущаются* продуктивными, пока важные вечно остаются в Idea. Поле **Impact** требует один раз письменно ответить «зачем это важно?» прежде, чем карточка пойдёт дальше.

### Помодоро попадает в окно концентрации при ADHD

Внимание при ADHD не отсутствует — оно *переменное*. Сессия в 25 минут соответствует реальному окну сфокусированного внимания для большинства людей с ADHD. Таймер встроен в каждую карточку — один клик запускает сессию, macOS-уведомление сообщает о конце.

### Маленькие шаги строят momentum (кайдзен)

Подзадачи с живым прогресс-баром превращают большие парализующие карточки в управляемое: разбей на 3–5 шагов и отмечай по одному. Когда последняя подзадача выполнена, карточка автоматически переходит в Done.

## Функции

| Функция | Почему помогает при ADHD |
|---|---|
| **Канбан-доска** (Idea → To Do → In Progress → Done) | Всё на одном экране — ничего не скрыто |
| **Drag & drop** | Физический жест = дофаминовая награда за продвижение задачи |
| **Поле Impact + бейдж KAIZEN** | Заставляет сформулировать *зачем* задача нужна, прежде чем браться за неё |
| **Подзадачи + прогресс-бар** | Разрушает паралич — большие задачи становятся маленькими шагами |
| **Авто-перемещение в Done при 100%** | Убирает шаг принятия решения, мгновенная награда за завершение |
| **Таймер Pomodoro (25/5 мин)** | Структурированные сессии, соответствующие окну концентрации при ADHD |
| **Бейджи дедлайна** (серый → жёлтый → красный пульс) | Визуальная срочность без тревоги от календаря |
| **Фильтр по приоритету** (High / Normal / Low) | Один клик — видишь только самое важное |
| **Теги + фильтр по тегам** | Группируй связанные задачи без вложенных папок |
| **Полнотекстовый поиск** | Найти что угодно мгновенно — память не нужна |
| **Сворачиваемые колонки** | Скрой шум Done/Archive, сосредоточься на активном |
| **Тёмная / светлая тема** | Снижает нагрузку на глаза при долгих сессиях |
| **Автозапуск при входе в систему** | Приложение всегда готово — никаких лишних действий |
| **Архив** | Убирай выполненные карточки с глаз, не удаляя их |
| **Ретро-панель** | Еженедельный обзор того, что реально сделано |
| **Локальные данные, без аккаунта** | Работает офлайн, без экрана входа, запускается мгновенно |

## Установка

### Скачать (рекомендуется)

Скачай последний `.dmg` на странице [Releases](https://github.com/IgorWarenik/adhd-kanzen-board/releases), открой его и перетащи **Kanzen Board.app** в папку Applications.

> **Важно: приложение не подписано.**
> У разработчика нет платного аккаунта Apple Developer ($99/год), поэтому DMG собирается без сертификата подписи кода. При первом запуске macOS Gatekeeper покажет предупреждение.
>
> **Как открыть несмотря на предупреждение:**
> 1. Нажми на приложение правой кнопкой мыши → **Открыть**
> 2. В диалоге снова нажми **Открыть**
>
> Это нужно сделать только один раз. После этого приложение открывается как обычно.
>
> Либо через системные настройки: **Системные настройки → Конфиденциальность и безопасность** → прокрути вниз до заблокированного приложения → нажми **Всё равно открыть**.

## Разработка

```bash
git clone https://github.com/IgorWarenik/adhd-kanzen-board.git
cd adhd-kanzen-board
npm install
npm start
```

Требуется Node.js 18+ и npm 9+.

---

## Сборка

Каждый пуш в `master` запускает GitHub Actions, который собирает `.dmg` для macOS (universal, x64 + arm64).

Чтобы опубликовать GitHub Release со ссылкой на скачивание, запушь тег версии:

```bash
git tag v1.0.1
git push origin v1.0.1
```
